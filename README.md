本程序集合了封装代码，压缩包，自动生成ico文件
自动生成程序介绍

封装代码：
      本程序采用pyinstaller封装模组命令，各种服务模式应有尽有

ico文件生成：
	本程序有单独的ico文件生成模块，也有在pyinstaller模式中如有没有ico文件，将提醒是否进入自动生成ico文件模式，需要有原图文件，生成后将自动直接封装程序

压缩包：
     本程序将封装好的文件都自动压缩到一个文件中，好发给别的人使用或者别的win平台上运行，增加便利性
